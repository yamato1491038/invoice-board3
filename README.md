# Name
  Invoive-board

## Description
  フィールド業務の成果や交通費をまとめるWEBアプリ
  業務委託で現場の成果をアプリに入力すると管理者がそれをまとめて成果に対して支払額を確認するもの

## Requirement
  Ruby 2.6.5

  Rails 6.0.0

  JQuery

  Node.js

  Pop.js
